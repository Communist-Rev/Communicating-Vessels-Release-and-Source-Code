Communicating Vessels 0.9 BETA

Инструкции по использованию программы:

Главное окно:

Для создания жидкостей необходимо нажать на кнопку "Добавить жидкость", добавление первой жидкости активирует кнопку "СТАРТ". До нажатия на неё не отображаются анимации, что позволяет добавить жидкости в нужном количестве и порядке. После начинается отрисовка движения согласно физическим законам, в это время блокируются кнопки добавления, сброса и вызова справки.
Время движения не соответствует реальному, все жидкости несмешиваемые по умолчанию (приближение к реальному времени и смешивание могут быть добавлены в следующих патчах).
После установления равновесия и перехода всех более плотных жидкостей вниз анимация останавливается. 
В любой момент остановить изображение можно кнопкой "СТОП", после чего возобновить кнопкой "СТАРТ".

Добавление жидкости:

Для добавления новой жидкости необходимо указать объём, плотность, цвет и сосуд, в который она будет помещена. Если какой-либо из параметров не заполнен, программа выдаст предупреждение. Также предупреждение выдаётся при превышении объёма в одном сосуде: максимальное значение - 4 литра в каждом. Ограничение плотности - 99999 кг/м³. В поля объёма и плотности вводить можно только целые числа.
В выпадающем списке существующих шаблонов можно выбрать один из готовых вариантов, а также создать свой шаблон или удалить один из существующих. Все шаблоны сохраняются в файл "default_liquids.txt". Выбранный вариант активируется нажатием на кнопку "применить".

Создание шаблона:

Для создания шаблона необходимо выбрать в выпадающем списке соответствующий вариант и нажать кнопку "применить". Окно создания шаблона работает по тому же принципу что и добавление жидкости, но помимо других параметров необходимо указать также название создаваемой жидкости.

Удаление шаблона:

Для удаления также необходимо выбрать соответствующий вариант в списке и нажать "применить". Далее нужно выбрать название шаблона, который требуется удалить и нажать кнопку "удалить".

Bug Report:

Обо всех багах можно сообщить автору приложения.
